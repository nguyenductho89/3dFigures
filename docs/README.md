# 3D Figure Scanner - Documentation

## Overview

This folder contains all product documentation for the 3D Figure Scanner iOS app - an application that uses LiDAR technology on iPhone 12 Pro and later to scan faces and bodies for 3D printing.

## Project Summary

- **Total User Stories**: 23
- **Total Story Points**: 122
- **Planned Sprints**: 6 (2 weeks each)
- **Team Velocity**: 25 points/sprint
- **Target Devices**: iPhone 12 Pro, 13 Pro, 14 Pro, 15 Pro (with LiDAR)

## Tech Stack

- **Platform**: iOS 16.0+
- **Language**: Swift 5.9+
- **Frameworks**: ARKit, RealityKit, Metal, SwiftUI
- **Architecture**: MVVM + Clean Architecture
